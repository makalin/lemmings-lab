Lemmings 2D

Controls

Use mouse to select and assign skills to lemmings

Skills: D Digger B Builder C Climber X Blocker

Press ESC to pause the game

Game Features

Guide the lemmings to the yellow exit point!

Each level has a time limit and requires saving a specific number of lemmings

Limited number of skills available per level

Dangers

⚠️ Water: Lemmings will drown if they fall in

⚠️ Traps: Spikes will kill lemmings on contact

Skills

Digger: Creates a path downward through terrain

Builder: Creates a bridge upward

Climber: Can climb over walls

Blocker: Stops and blocks other lemmings
